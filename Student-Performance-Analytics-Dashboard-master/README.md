# 🎓 Student Performance Analytics Dashboard

An interactive dashboard that helps educational institutions **identify at-risk students** by analyzing marks, attendance, and login behavior.

> 🚀 Developed as part of the **RISE Internship Program (Tamizhan Skills)**

---

## 🧠 Objective

Institutions often need **early warning systems** to identify students who might fail or drop out.  
This project analyzes student data to highlight:

- Performance trends
- Risk areas
- Impact of absenteeism

---

## 📊 Features

✅ **Performance Analysis**
- Average marks, attendance, and login frequency
- Comparison between top and struggling students

✅ **Correlation Insights**
- Heatmap showing relationships between attendance, marks, and logins

✅ **Absentee Impact**
- Visual trends showing how attendance affects academic performance

✅ **Dashboard Visuals**
- Bar charts for student comparison
- Correlation heatmap
- Personalized export (optional)

✅ **Ready for Academic Use**
- Helps teachers or advisors make **data-driven decisions**

---

## 📂 Dataset Used

The dataset contains:
- `Student Name`
- `Marks`
- `Attendance (%)`
- `Login Count`

(A sample CSV is included in this repo)
